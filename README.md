# Retro Blocks

Universal web components library with retro styling built with Lit.

## Development

```bash
# Install dependencies
npm install

# Start Storybook
npm run storybook

# Build library
npm run build
```

## Project Structure

```
src/
├── components/     # Lit web components
├── styles/         # Shared styles and themes
├── stories/        # Storybook stories
├── .storybook/     # Storybook configuration
└── index.ts        # Main entry point
```
